# Infosys AI Agent Project
AI-powered automated web testing agent built as part of the Infosys Internship.

## Milestone 1 – Completed
- Flask web application with Home, Login, and Search pages
- Playwright automated browser test
- LangGraph workflow stub (NL instruction → action plan)
- REST API endpoint: `/api/submit_test`
- Unit and integration tests using pytest

## Tech Stack
- Python 3.11
- Flask
- Playwright
- LangGraph
- Pytest
