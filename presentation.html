2. Code Generator Module

🔧 Key Functions

  • generate_playwright_code()
    Main function - builds complete script
  • _get_url_for_page()
    Maps page names to URL paths
  • _get_selector_for_field()
    Maps fields to CSS selectors
  • _get_selector_for_button()
    Maps buttons to selectors
  • generate_assertions()
    Creates validation statements

📤 Generated Output

from playwright.sync_api import sync_playwright

def run_test():
  with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    page.goto("http://localhost:5000/")
    page.click('button:has-text("Login")')
    browser.close()

3. LangGraph Workflow

🔗 StateGraph Pipeline

class AgentState(TypedDict):
  instruction: str
  parsed_actions: List[Dict]
  generated_code: str
  assertions: List[Dict]
  status: str

# Workflow definition
workflow = StateGraph(AgentState)
workflow.add_node("parse", parse_node)
workflow.add_node("generate_code", generate_code_node)
workflow.set_entry_point("parse")
workflow.add_edge("parse", "generate_code")
workflow.add_edge("generate_code", END)

📊 Workflow Nodes

  • Entry Point: parse
  • Node 1: parse_node()
    Calls instruction_parser.py
  • Node 2: generate_code_node()
    Calls code_generator.py
  • End: Returns final state with code
parse
→
generate_code
→
END
