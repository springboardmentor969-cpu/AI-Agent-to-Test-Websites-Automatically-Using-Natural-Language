================================================================================
                    MODULE 1: INSTRUCTION PARSER
                    (agent/instruction_parser.py)
================================================================================

PURPOSE:
--------
Parses natural language test instructions and extracts structured browser 
actions that can be understood by the code generator.

MAIN FUNCTION:
--------------
  parse_instruction(instruction: str) -> list

INPUT EXAMPLE:
--------------
  "Open home page and click login button"

OUTPUT EXAMPLE:
---------------
  [
    {"action": "open_page", "target": "home page"},
    {"action": "click", "target": "login button"}
  ]

SUPPORTED ACTIONS:
------------------
  1. NAVIGATION: open, navigate, go to
     - Detects: home, login, dashboard, profile, settings pages
     - Also supports direct URLs (http://...)
     
  2. TEXT INPUT: enter, type, fill
     - Detects: username, password, email, search fields
     - Extracts values from quotes in instruction
     
  3. CLICK: click, submit, press
     - Detects: submit, login, save, search, register buttons
     
  4. VERIFY: verify, check, assert
     - Detects: error message, success message, page title, text
     
  5. WAIT: wait X seconds
     - Extracts duration from instruction

HOW IT WORKS:
-------------
  1. Convert instruction to lowercase for pattern matching
  2. Use regex to extract URLs and quoted values
  3. Check for action keywords (open, click, enter, etc.)
  4. Map detected elements to structured action dictionaries
  5. Return list of actions in execution order

================================================================================
                     MODULE 2: CODE GENERATOR
                     (agent/code_generator.py)
================================================================================

PURPOSE:
--------
Converts structured actions from the parser into executable Playwright Python 
scripts that can be run for automated browser testing.

MAIN FUNCTIONS:
---------------
  1. generate_playwright_code(parsed_actions, target_url) -> str
  2. generate_assertions(parsed_actions) -> list

HELPER FUNCTIONS:
-----------------
  - _get_url_for_page(page_name, base_url)
      Maps page names to URL paths
      Example: "home page" -> "/"
               "login page" -> "/login"
               
  - _get_selector_for_field(field_name)
      Maps field names to CSS selectors
      Example: "username field" -> 'input[name="username"]'
      
  - _get_selector_for_button(button_name)
      Maps button names to CSS selectors
      Example: "login button" -> 'button:has-text("Login")'

GENERATED CODE EXAMPLE:
-----------------------
  from playwright.sync_api import sync_playwright

  def run_test():
      with sync_playwright() as p:
          browser = p.chromium.launch(headless=True)
          page = browser.new_page()
          
          page.goto("http://localhost:5000/")
          print("Navigated to: http://localhost:5000/")
          
          page.click('button:has-text("Login")')
          print("Clicked: login button")
          
          browser.close()
          print("Test completed successfully!")

  if __name__ == "__main__":
      run_test()

ASSERTIONS OUTPUT:
------------------
  [
    {
      "type": "page_loaded",
      "description": "Page 'home page' should load successfully",
      "check": "page.url contains expected path"
    },
    {
      "type": "element_clickable", 
      "description": "'login button' should be clickable",
      "check": "element is visible and enabled"
    }
  ]

================================================================================
                     MODULE 3: LANGGRAPH AGENT
                     (agent/langgraph_agent.py)
================================================================================

PURPOSE:
--------
Orchestrates the workflow pipeline using LangGraph's StateGraph to connect
the instruction parser and code generator in a modular, extensible way.

STATE DEFINITION:
-----------------
  class AgentState(TypedDict):
      instruction: str           # Original user input
      parsed_actions: List[Dict] # Output from parser
      generated_code: str        # Output from code generator
      assertions: List[Dict]     # Generated test assertions
      status: str                # Workflow status
      error: str                 # Error message if any

WORKFLOW NODES:
---------------
  Node 1: parse_node
    - Input: AgentState with instruction
    - Calls: parse_instruction() from instruction_parser.py
    - Output: AgentState with parsed_actions
    
  Node 2: generate_code_node
    - Input: AgentState with parsed_actions
    - Calls: generate_playwright_code() and generate_assertions()
    - Output: AgentState with generated_code and assertions

WORKFLOW PIPELINE:
------------------
  [Entry] -> parse_node -> generate_code_node -> [END]

MAIN ENTRY POINT:
-----------------
  handle_instruction(instruction: str) -> Dict

  This function:
    1. Initializes the state with the user instruction
    2. Invokes the compiled workflow
    3. Returns the final result with code and assertions
